# MOSFET 開關溫度計算重點整理

> 適用工具：SIMetrix / Infineon L1 模型

---

## 一、L1 模型的限制

- L1 模型假設**固定溫度**，無法自動計算自加熱效果
- 溫度由模擬工具的全域設定控制（`.TEMP` 指令）
- 若需動態計算 Tj，應改用 **L3 模型**（有 Tj、Tcase 接腳）

---

## 二、L1 模型估算 Tj 的流程（迭代法）

```
設初始溫度（通常 25°C）
        ↓
    跑 SIMetrix 模擬
        ↓
    量測平均損耗 P
        ↓
    手算 Tj = T_amb + P × (Rth_jc + Rth_ca)
        ↓
    Tj 與上次差 > 5°C？→ 用新 Tj 重設 .TEMP 重跑
        ↓
    收斂 → 最終 Tj 即為估算結果
```

---

## 三、核心公式

### 3.1 損耗計算

| 損耗類型 | 公式 |
|---------|------|
| 導通損耗 | `P_cond = I_rms² × RDS(on, T)` |
| 開關損耗 | `P_sw`（從 SIMetrix 電壓×電流波形積分量測） |
| 總損耗   | `P_total = P_cond + P_sw` |

### 3.2 RDS(on) 溫度修正

$$RDS(on, T) = RDS(on, 25°C) \times \left[1 + \frac{TC_{rds}}{100} \times (T - 25)\right]$$

- `TC_rds`：溫度係數，典型值約 **0.4 ~ 0.6 %/°C**（查 datasheet 圖）

### 3.3 接面溫度

$$T_j = T_{amb} + P_{total} \times (R_{th,jc} + R_{th,ca})$$

### 3.4 安全裕度

$$\text{Margin} = T_{j,max} - T_j$$

> 建議裕度至少保留 **20 ~ 30°C**

### 3.5 最低需求散熱器熱阻（反推）

$$R_{th,ca,min} = \frac{T_{j,max} - T_{amb}}{P_{total}} - R_{th,jc}$$

---

## 四、知道 Tj 有什麼用

| 用途 | 說明 |
|------|------|
| **安全判斷** | 確認 `Tj < Tj,max`（通常 150°C 或 175°C） |
| **損耗修正** | RDS(on) 隨溫度上升，迭代才能得到準確損耗值 |
| **散熱器選型** | 反推 Rth(c→a) 需求，決定散熱器規格或是否需要風扇 |
| **可靠度評估** | 溫度每升高 10°C，壽命約減半（Arrhenius 法則） |

---

## 五、已知 Tj 後，換算新環境溫度的 Tj

### 快速平移估算（簡便法）

$$T_{j,new} = T_{j,old} + (T_{amb,new} - T_{amb,old})$$

> **注意**：環境溫度升高時，RDS(on) 也會增加，損耗 P 同步上升，  
> 嚴格應**重新迭代**，快速估算才用平移公式。

### 驗算散熱設計是否仍足夠

$$R_{th,ca} \leq \frac{T_{j,max} - T_{amb,new}}{P} - R_{th,jc}$$

## 六、L1 vs L3 模型比較

| 需求 | 建議模型 |
|------|---------|
| 效率 / 開關損耗分析（固定溫度） | **L1** |
| 動態 Tj 監測、自加熱效應 | **L3** |
| 熱管理設計、散熱器建模 | **L3** |
